<a id="install">
  <h2>Install</h2>
</a>
<h3> Make </h3>

```
$ git clone https://github.com/endygamedev/party-parrot.git
$ cd party-parrot
$ sudo make install
```

<a id="uninstall">
  <h2>Uninstall</h2>
</a>
<h3> Make </h3>

```
$ sudo make uninstall
```

<a id="update">
  <h2>Update</h2>
</a>
<h3> Make </h3>

```
$ sudo make update
```

<a id="usage">
  <h2>Usage</h2>

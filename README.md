<a id="install">
  <h2>Instal</h2>
</a>
<h3> Make </h3>

```
$ git clone https://github.com/endygamedev/party-parrot.git
$ cd party-parrot
$ sudo make install
```

<a id="uninstal">
  <h2>Uninstal</h2>
</a>
<h3> Make </h3>

```
$ sudo make uninstall
```

<a id="update">
  <h2>Update</h2>
</a>
<h3> Make </h3>

```
$ sudo make update
```

<a id="usage">
  <h2>Usage</h2>
</a>

```

```

<h3>Keys</h3>

<h4>Change encoded image</h4>

```
$ parrot -c ascii
```
